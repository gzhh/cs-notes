# Go command
- https://go.dev/doc/cmd
- https://pkg.go.dev/cmd/go

## Compile and run go program
运行指定文件
- `go run main.go`

运行指定包下多个文件的方式
- `go run .`
- `go run *.go`

Go test 相关
- `go test match`
- `go test ./...`
- `go test .`
