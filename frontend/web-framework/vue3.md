# Vue3


## 创建 vue3 项目

1.使用 npm

```
npm create vue@latest
cd vue-project
npm install
npm run format
npm run dev
```

2.使用 Vue CLI

```
npm install -g @vue/cli
vue create my-project
cd my-project
npm run serve
```

3.使用 Vite

```
npm create vite@latest my-project --template vue
cd my-project
npm install
npm run dev
```
