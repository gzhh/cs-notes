# Python
## pyenv (Python version management)
- https://github.com/pyenv/pyenv

### Usage
List all Python versions available to pyenv
- `pyenv versions`

Install a Python version
- `pyenv install 3.10.4`

Set the global Python version
- `pyenv global 3.10.4`
